Git的使用方法

Git既可以用于本地管理代码的版本更改、回溯，同时也可以远程拉取仓库。

1.本地同一个分支使用的git命令

git init:新建一个仓库

git add xxxxxx.js:添加进暂存区

git commit -m “需要提交的信息”:提交

git log :查看提交记录(Page键翻页查看，按q退出)

git reflog：查看命令历史

git reset –hard (commit_id)

2.分支的命令

git switch -c dev 创建并切换到dev分支

git switch dev 切换到dev分支

git branch 查看所有分支

git merge dev 合并分支

git branch -d dev 删除分支 git branch -D dev 强制删除分支
